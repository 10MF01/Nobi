# Nobi（のびちゃん）

一款桌面学习监督宠物应用。桌面角色 **のびちゃん**（取自日语「伸びる」，意为持续成长）常驻桌面，陪伴并监督你的每日学习/工作计划，通过动态表情动画 + 可自定义的鼓励/治愈/批评文案给你情感反馈，帮助你保持自律、坚持长期目标。

个人使用工具，纯本地单用户，不含账号系统/云同步/后端服务，所有数据都保存在本机。

## 功能

**桌面悬浮角色**
- 手绘 SVG 角色，`framer-motion` 驱动的呼吸/摇摆/跳动动画，idle/happy/comfort/stern 四种情绪状态
- 可拖拽到桌面任意位置；单击摸一下会有小反应，双击打开设置面板
- 常驻置顶、不占任务栏，支持全屏应用上方显示

**计划管理**（面板"计划管理"页）
- 每日任务、每周任务（可选星期几）、长期目标/考证倒计时、临时待办，四种计划类型
- 打卡/完成勾选，每周任务只在到期的星期几可打卡
- 增删改查，行内编辑

**情绪反馈与文案库**（面板"文案库"页）
- 打卡/完成目标后，のびちゃん 会根据当日完成率、连续达标天数，播放对应情绪动画并弹出一条气泡文案（鼓励/治愈/批评/庆祝）
- 5 类文案共 35 条种子内容，全部可在面板里增删改、启用/停用，避免重复播放同一句

**主动提醒与日终总结**（面板"提醒设置"页）
- 中午/晚间打卡提醒：今天还有没打卡的任务时才轻推一下，全部打卡完就安静不打扰
- 日终总结：定时计算当天完成率和连续天数，播放反应并弹出系统通知
- 三个时间点均可自定义开关和时刻，支持"立即测试"手动触发，不用等到设定时间

**历史记录**（面板"历史记录"页）
- 当前连续达标天数、过去 42 天平均完成率
- 7 列日历热力图，按完成率显示颜色深浅，"当天没有安排任务"和"当天 0% 完成"用不同样式区分

**应用设置**（面板"应用设置"页）
- 开机自启动开关
- のびちゃん 形象大小（60%~180%）、悬浮窗透明度（30%~100%），拖动滑块实时预览生效

**跨平台打包**
- Windows（nsis 安装包）+ macOS（dmg）双平台目标

## 技术栈

- Electron + React + TypeScript（`electron-vite` 脚手架）
- `framer-motion` 驱动的手绘 SVG 角色动画
- `better-sqlite3` 本地数据存储
- Ant Design（面板 UI）
- `node-schedule`（定时提醒）

## 开发

```bash
npm install              # 安装依赖
npm run dev              # 启动开发模式（带 HMR）
npm run typecheck        # 类型检查
npm run lint             # eslint
npm run build:win        # 打包 Windows 安装包
npm run build:mac        # 打包 macOS
```

详细架构说明见 [`CLAUDE.md`](./CLAUDE.md)。
